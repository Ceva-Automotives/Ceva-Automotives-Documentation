# Épicos, Features e Histórias de Usuário - Ceva Automotives

| **Épico** | **Feature** | **ID** | **História de Usuário** | **Critérios de Aceitação** |
|-----------|-------------|--------|-------------------------|----------------------------|
| **EP-01: Gestão Administrativa** | **FT-01: Gestão de Veículos** | US-01 | **Como** Administrador, **eu quero** adicionar novos veículos à frota, **para que** possamos expandir nossas ofertas. | - Sistema deve permitir cadastro de diferentes tipos (SUV, Conversível, Elétrico ...)<br>- Todos os campos obrigatórios devem ser preenchidos<br>- Veículo deve ter status "Disponível" por padrão<br>- Validação de placa única no sistema |
| | | US-02 | **Como** Administrador, **eu quero** editar informações de veículos existentes, **para que** os dados da frota estejam atualizados. | - Busca por placa ou modelo deve funcionar<br>- Todas as informações, exceto placa, devem ser editáveis|
| | | US-03 | **Como** Administrador, **eu quero** remover veículos da frota, **para que** possamos gerenciar nossos ativos eficazmente. | - Veículo só pode ser removido sem reservas ativas/futuras<br>- Status deve ser alterado para "Inativo"<br>- Veículo não deve aparecer no catálogo<br>- Confirmação obrigatória antes da remoção |
| | | US-04 | **Como** Administrador, **eu quero** alterar status de veículos, **para que** eu possa controlar disponibilidade. | - Status alterado automaticamente para "Locado" em reservas<br>- Alteração manual para "Manutenção" ou "Disponível"<br>- Veículo em "Manutenção" não pode ser reservado |
| | **FT-02: Gestão de Clientes** | US-05 | **Como** Administrador, **eu quero** visualizar lista de clientes cadastrados, **para que** eu tenha visão geral da base. | - Lista com informações básicas (nome, e-mail, telefone)<br>- Busca por nome ou CPF|
| | | US-06 | **Como** Administrador, **eu quero** visualizar detalhes de clientes, **para que** eu possa oferecer melhor atendimento. | - Tela com todas as informações do cliente<br>- Histórico completo de locações<br>- Acesso aos detalhes de cada reserva<br>- Informações de contato atualizadas |
| | | US-07 | **Como** Administrador, **eu quero** desativar contas de clientes, **para que** possamos gerenciar usuários com maior eficácia. | - Cliente com reservas ativas não pode ser desativado<br>-Possibilidade de reativar conta |
| | **FT-03: Gestão de Reservas** | US-08 | **Como** Administrador, **eu quero** visualizar todas as reservas, **para que** eu possa gerenciar o processo de locação. | - Lista filtrada por status (Pendente, Confirmada, etc.)<br>- Busca por cliente ou veículo<br>- Ordenação por data de criação/início |
| | | US-09 | **Como** Administrador, **eu quero** confirmar ou cancelar reservas pendentes, **para que** clientes sejam notificados. | - Confirmação altera status do veículo para "Locado"<br>- Cancelamento libera veículo<br>- Notificação por e-mail ao cliente<br>- Registro de motivo do cancelamento |
| | **FT-04: Relatórios e Dashboard** | US-10 | **Como** Administrador, **eu quero** visualizar relatório de veículos mais locados, **para que** eu possa otimizar a frota. | - Ranking de veículos por número de locações<br>- Filtro por período e categoria<br>- Análise de receita por veículo<br>- Identificação de veículos ociosos |
| **EP-02: Experiência do Cliente** | **FT-05: Autenticação e Perfil** | US-11 | **Como** visitante, **eu quero** me cadastrar no sistema, **para que** eu possa alugar veículos. | - Validação de CPF e e-mail<br>- E-mail de confirmação para ativar conta |
| | | US-12 | **Como** Cliente, **eu quero** fazer login na conta, **para que** eu possa acessar funcionalidades. | - Login com e-mail e senha<br>- Opção "Esqueci minha senha"<br>- Bloqueio após 3 tentativas incorretas |
| | | US-13 | **Como** Cliente, **eu quero** visualizar e editar meu perfil, **para que** meus dados estejam atualizados. | - Alteração de endereço, telefone e senha<br>- Confirmação para mudança de e-mail|
| | **FT-06: Catálogo de Veículos** | US-14 | **Como** Cliente, **eu quero** visualizar catálogo de veículos, **para que** eu possa escolher um para alugar. | - Exibição de foto, marca, modelo e valor diário<br>- Detalhes ao clicar no veículo<br>- Apenas veículos disponíveis no catálogo<br>- Informações de características principais |
| | | US-15 | **Como** Cliente, **eu quero** filtrar e ordenar o catálogo, **para que** eu encontre o carro ideal. | - Filtros: categoria, tipo, características<br>- Ordenação por preço (crescente/decrescente)<br>- Filtro por disponibilidade em período<br>- Busca por marca/modelo |
| | **FT-07: Processo de Reserva** | US-16 | **Como** Cliente, **eu quero** selecionar veículo e período para reservar, **para que** eu garanta disponibilidade. | - Seleção de data/hora retirada e devolução<br>- Cálculo e exibição do valor total<br>- Escolha entre tipos de seguro<br>- Validação de disponibilidade em tempo real |
| | | US-17 | **Como** Cliente, **eu quero** receber confirmação por e-mail, **para que** eu tenha comprovante da reserva. | - E-mail com detalhes completos da reserva |
| | **FT-08: Gestão de Reservas do Cliente** | US-18 | **Como** Cliente, **eu quero** visualizar histórico de locações, **para que** eu possa acompanhar reservas. | - Lista de todas as reservas (passadas e futuras)<br>- Visualização de detalhes de cada reserva<br>- Acesso fácil a partir do painel<br>- Status atualizado em tempo real |
| | | US-19 | **Como** Cliente, **eu quero** cancelar reserva futura, **para que** eu tenha flexibilidade. | - Cancelamento apenas para reservas "Confirmadas"<br>- Antecedência mínima de 48 horas<br>- Informação sobre taxas de cancelamento<br>- Notificação por e-mail após cancelamento |
| | | US-20 | **Como** Cliente, **eu quero** avaliar veículo após locação, **para que** eu possa compartilhar experiência. | - Sistema de avaliação por estrelas (1-5)<br>- Campo para comentários opcionais<br>- Avaliação apenas após finalização da reserva<br>- Exibição de avaliações no catálogo |
| **EP-03: Operações e Suporte** | **FT-09: Notificações** | US-21 | **Como** sistema, **eu quero** enviar notificações automáticas, **para que** usuários sejam informados sobre eventos importantes. | - E-mail de confirmação de reserva<br>- Notificação de cancelamento |
| | | US-22 | **Como** Administrador, **eu quero** gerenciar usuários do sistema, **para que** eu possa controlar acessos. | - Criação de contas de administrador<br>- Definição de níveis de permissão<br>- Auditoria de ações dos usuários<br>- Bloqueio/desbloqueio de contas |
